JSONP概念
由于同源策略，一般来说不允许JavaScript跨域访问其他服务器的页面对象，但是HTML的script元素是一个例
外。利用 script 元素的这个开放策略，网页可以得到从其他来源动态产生的 JSON资料，而这种使用模式就是所
谓的 JSONP。用 JSONP 抓到的资料并不是 JSON，而是任意的JavaScript，用 JavaScript 直译器执行而不是用 JSON 解析器解析
